# ===== EN =====

## Personal Portfolio Website Documentation

## Overview

This is a responsive personal portfolio website template built with Bootstrap 5.3 and jQuery. The template includes multiple sections to showcase personal information, skills, projects, employment history, and client testimonials.

## File Structure

- `index.html`: Main HTML file containing the website structure
- `style.css`: Custom CSS styles for the website
- `functionality.js`: JavaScript for carousel functionality

## Sections

### 1. Navigation Bar

- Fixed at the top of the page
- Dark theme with smooth scrolling navigation links
- Links to all main sections of the page

### 2. Personal Introduction

- Name and title display
- Short description paragraph
- Download CV button with icon
- Profile image placeholder

### 3. About Me

- Detailed personal description in multiple paragraphs
- Background information and professional summary

### 4. Education & Skills

#### Education

- List of educational background with:
  - University names
  - Majors
  - Years attended

#### Soft Skills

- List of personal/professional skills

### 5. Technical Skills

- Interactive carousel displaying technical skills
- Each skill includes:
  - Image placeholder
  - Title
  - Short description
- Navigation arrows for carousel control
- Responsive design (shows fewer items on mobile)

### 6. Projects

#### Current Projects

- Single project displayed with:
  - Project name
  - "View" button with icon
  - Description

#### Past Projects

- Three projects listed with:
  - Project names
  - "View" buttons with icons
  - Descriptions

### 7. Employment History & Contact

#### Employment History

- Two previous jobs listed with:
  - Company names
  - Employment dates
  - Job descriptions

#### Socials & Contact

- Contact information with icons:
  - Facebook link
  - LinkedIn link
  - Email address
  - Phone number

### 8. Client Commentaries

- Four client testimonials displayed in cards
- Each includes:
  - Client name
  - Job title
  - Testimonial text
- Hover effects on cards

### 9. Footer

- Copyright information
- "Back to top" link

## Technical Features

### Responsive Design

- Mobile-first approach
- Media queries for different screen sizes
- Carousel adjusts number of visible items based on screen width

### Interactive Elements

- Smooth scrolling navigation
- Hover effects on:
  - Project cards
  - Client testimonial cards
  - Buttons
- Functional carousel with navigation controls

### Styling

- Custom CSS overriding some Bootstrap defaults
- Consistent color scheme
- Icon integration (Bootstrap Icons)
- Card styling with shadows and transitions

### JavaScript Functionality

- Carousel item cloning to ensure smooth transitions
- Responsive carousel behavior

## Dependencies

- Bootstrap 5.3 (CSS and JS)
- jQuery 3.6.4
- Bootstrap Icons (embedded in SVG format)

## Customization Notes

To personalize this template:

1. Replace placeholder images with actual photos
2. Update personal information in all sections
3. Add real project details and links
4. Include genuine client testimonials
5. Connect social media links to your profiles
6. Replace the CV download with your actual resume

The template is designed to be easily customizable while maintaining its responsive design and functionality.

- [Dark version](https://vxdpxl.com.mx/personal-page/personal-page-dark)

# ===== ES =====

## Documentación del Sitio Web de Portafolio Personal

## Visión General

Esta es una plantilla de sitio web de portafolio personal responsive construida con Bootstrap 5.3 y jQuery. La plantilla incluye múltiples secciones para mostrar información personal, habilidades, proyectos, historial laboral y testimonios de clientes.

## Estructura de Archivos

- `index.html`: Archivo HTML principal que contiene la estructura del sitio
- `style.css`: Estilos CSS personalizados para el sitio web
- `functionality.js`: JavaScript para la funcionalidad del carrusel

## Secciones

### 1. Barra de Navegación

- Fija en la parte superior de la página
- Tema oscuro con enlaces de navegación de desplazamiento suave
- Enlaces a todas las secciones principales de la página

### 2. Introducción Personal

- Visualización del nombre y título
- Párrafo breve de descripción
- Botón para descargar CV con icono
- Espacio reservado para imagen de perfil

### 3. Sobre Mí

- Descripción personal detallada en múltiples párrafos
- Información de antecedentes y resumen profesional

### 4. Educación y Habilidades

#### Educación

- Lista de formación académica con:
  - Nombres de universidades
  - Carreras
  - Años de asistencia

#### Habilidades Blandas

- Lista de habilidades personales/profesionales

### 5. Habilidades Técnicas

- Carrusel interactivo que muestra habilidades técnicas
- Cada habilidad incluye:
  - Espacio reservado para imagen
  - Título
  - Descripción breve
- Flechas de navegación para control del carrusel
- Diseño responsive (muestra menos elementos en móviles)

### 6. Proyectos

#### Proyectos Actuales

- Proyecto único mostrado con:
  - Nombre del proyecto
  - Botón "Ver" con icono
  - Descripción

#### Proyectos Anteriores

- Tres proyectos listados con:
  - Nombres de proyectos
  - Botones "Ver" con iconos
  - Descripciones

### 7. Historial Laboral y Contacto

#### Historial Laboral

- Dos trabajos anteriores listados con:
  - Nombres de empresas
  - Fechas de empleo
  - Descripciones del puesto

#### Redes Sociales y Contacto

- Información de contacto con iconos:
  - Enlace a Facebook
  - Enlace a LinkedIn
  - Dirección de correo
  - Número de teléfono

### 8. Comentarios de Clientes

- Cuatro testimonios de clientes mostrados en tarjetas
- Cada uno incluye:
  - Nombre del cliente
  - Puesto de trabajo
  - Texto del testimonio
- Efectos hover en las tarjetas

### 9. Pie de Página

- Información de copyright
- Enlace "Volver arriba"

## Características Técnicas

### Diseño Responsive

- Enfoque mobile-first
- Media queries para diferentes tamaños de pantalla
- El carrusel ajusta el número de elementos visibles según el ancho de pantalla

### Elementos Interactivos

- Navegación con desplazamiento suave
- Efectos hover en:
  - Tarjetas de proyectos
  - Tarjetas de testimonios
  - Botones
- Carrusel funcional con controles de navegación

### Estilos

- CSS personalizado que sobrescribe algunos valores por defecto de Bootstrap
- Esquema de colores consistente
- Integración de iconos (Bootstrap Icons)
- Estilo de tarjetas con sombras y transiciones

### Funcionalidad JavaScript

- Clonación de elementos del carrusel para asegurar transiciones suaves
- Comportamiento responsive del carrusel

## Dependencias

- Bootstrap 5.3 (CSS y JS)
- jQuery 3.6.4
- Bootstrap Icons (incrustado en formato SVG)

## Notas de Personalización

Para personalizar esta plantilla:

1. Reemplazar las imágenes de placeholder con fotos reales
2. Actualizar la información personal en todas las secciones
3. Añadir detalles y enlaces reales de proyectos
4. Incluir testimonios genuinos de clientes
5. Conectar enlaces de redes sociales a tus perfiles
6. Reemplazar la descarga de CV con tu currículum real

La plantilla está diseñada para ser fácilmente personalizable manteniendo su diseño responsive y funcionalidad.
